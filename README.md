# 🐍 ATILA-PYTHON

O **ATILA-PYTHON** é um repositório dedicado ao desenvolvimento de scripts automatizados, lógica de programação e ferramentas utilitárias utilizando a linguagem Python. O foco deste espaço é centralizar algoritmos de cálculo, manipulação de estruturas de dados e protótipos de interfaces de utilizador.

## 🛠️ Scripts e Funcionalidades

O projeto está organizado dentro da pasta `scripts/` e conta com os seguintes módulos utilitários:

* **`calcula_co...`:** Script focado em automação de cálculos ou processamento matemático de dados.
* **`interface.py`:** Protótipo inicial para interações de utilizador via terminal ou interface gráfica.
* **`list_pr.py`:** Algoritmos para manipulação, filtragem e organização de listas ou estruturas de dados.
* **`se_pr.py`:** Implementação de estruturas condicionais e fluxos de decisão lógicos.
